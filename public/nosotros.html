El proceso

Cómo funciona

Tres pasos simples separan un pedido de una entrega exitosa.

1
Paso 1

El pedido

El cliente elige el producto desde el catálogo. El vendedor o dropshipper confirma el pedido en el sistema con los datos de entrega.

2
Paso 2

El despacho

El proveedor prepara el producto y lo entrega al delivery asignado. El estado se actualiza en tiempo real para todos los involucrados.

3
Paso 3

La entrega

El delivery lleva el paquete hasta la puerta del cliente. El cliente puede rastrear su pedido con su código de seguimiento.
